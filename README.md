
Based on Moco framework(https://github.com/dreamhead/moco).

## Quick Start

* Clone Moco

```shell
git clone https://github.com/xuyaozhong/moco1400.git
```
* Build Moco

```shell
./gradlew build
```
* Build uberjar

```shell
./gradlew uberjar
```
* Check code before commit

```shell
./gradlew check
```
